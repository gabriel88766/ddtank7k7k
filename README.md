# This is a modification of AsteroidApp 
link: (https://github.com/asteroid-dev/AsteroidApp/)

The alternative browser with Flash Player for Windows 7/10/11 (x32/x64), MacOS, and now Linux.

## Como instalar
1. Download NodeJS: https://nodejs.org/
 
2. Abra o `Prompt de Comando`, execute git clone no repositório e entre no diretório DDTank7k7k.

3. Instale as dependências executando o comando: `npm install`

4. Execute o comando: `npm start` 

5. Por último, execute
    * `npm run windows` - Para Windows 
    * `npm run mac` - Para Mac 
    * `npm run linux` - Para Linux 
    
6. O launcher estará na pasta dist. Você pode executar o app ou instalar, como desejar.
